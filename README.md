# تطبيق ذكرني (Thakarni)

**ذكرني** هو تطبيق إسلامي شامل يهدف إلى أن يكون رفيقك اليومي في العبادة والذكر. يتميز التطبيق بتصميم عصري وفخم (Modern & Luxury Design) يجمع بين جمالية الواجهة وروحانية المحتوى.

## المميزات الحالية
- **الأذكار اليومية:** أذكار الصباح والمساء وما بعد الصلاة.
- **القرآن الكريم:** مصحف كامل للقراءة.
- **مواقيت الصلاة:** مواقيت دقيقة مع عد تنازلي للصلاة القادمة.
- **أسماء الله الحسنى:** عرض الأسماء مع معانيها للتدبر.
- **التسبيح الذكي:** مسبحة إلكترونية تفاعلية.
